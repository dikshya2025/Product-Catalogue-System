Project Description

The Secure Retail Product Catalogue System allows administrators to manage product listings while providing regular users with a secure browsing experience. The system includes user authentication, role-based access control, and input validation.

Problem Statement

Many retail systems lack proper security controls, leaving product data vulnerable to unauthorised access and attacks. This project addresses these issues by implementing secure coding practices throughout the development lifecycle.

Our Team

We are a group of 6 students collaborating on this capstone project, each contributing through defined roles including Project Manager, Developers, Tester, and Reviewers.
